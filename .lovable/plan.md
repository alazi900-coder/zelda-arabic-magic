
# دعم ألعاب متعددة — كل لعبة بمسار مستقل

## الفكرة العامة
تحويل الصفحة الرئيسية إلى **شاشة اختيار اللعبة**، مع نقل مسار زيلدا الحالي بالكامل تحت `/zelda/...` بدون تعديل أي من ملفاته الداخلية.

## التغييرات المطلوبة

### 1. صفحة رئيسية جديدة — اختيار اللعبة
- إنشاء صفحة `src/pages/GameSelect.tsx` تعرض بطاقات الألعاب المدعومة
- كل بطاقة تحتوي: اسم اللعبة، أيقونة، حالة الدعم (جاهز / قريباً)
- زيلدا تكون اللعبة الوحيدة "الجاهزة" حالياً، والباقي يظهر كـ "قريباً"

### 2. تعديل المسارات (Routes)
- الصفحة الرئيسية `/` تصبح شاشة اختيار اللعبة
- مسارات زيلدا الحالية تنتقل تحت `/zelda/`:
  - `/zelda` — الصفحة التعريفية الحالية (Index.tsx)
  - `/zelda/process` — صفحة المعالجة
  - `/zelda/editor` — المحرر
  - `/zelda/results` — النتائج
- ألعاب مستقبلية ستكون تحت مسارات مثل `/splatoon/`, `/mario/` إلخ

### 3. تعديل الروابط الداخلية
- تحديث روابط "العودة" في صفحات زيلدا لتشير إلى `/zelda` و `/zelda/process` بدلاً من `/` و `/process`
- تحديث زر "ابدأ التعريب" في Index.tsx ليشير إلى `/zelda/process`

### الملفات المتأثرة

| الملف | نوع التغيير |
|---|---|
| `src/pages/GameSelect.tsx` | **جديد** — صفحة اختيار اللعبة |
| `src/App.tsx` | تعديل المسارات فقط |
| `src/pages/Index.tsx` | تعديل الروابط (`/process` -> `/zelda/process`) |
| `src/pages/Process.tsx` | تعديل الروابط (`/` -> `/zelda`, `/editor` -> `/zelda/editor`) |
| `src/pages/Editor.tsx` | تعديل رابط العودة (`/process` -> `/zelda/process`) |
| `src/pages/Results.tsx` | تعديل الروابط |

### ما لن يتغير
- **لا تعديل** على Edge Functions (arabize, translate-entries, إلخ)
- **لا تعديل** على منطق المحرر أو الـ Hooks
- **لا تعديل** على معالجة النصوص العربية أو MSBT/SARC

### القسم التقني

```text
المسارات الجديدة:
/                    -> GameSelect (اختيار اللعبة)
/zelda               -> Index (صفحة زيلدا التعريفية)
/zelda/process       -> Process
/zelda/editor        -> Editor
/zelda/results       -> Results
/splatoon            -> (قريباً)
/mario               -> (قريباً)
```

صفحة اختيار اللعبة ستعرض بطاقات بتصميم مشابه للتصميم الحالي، مع ألوان مميزة لكل لعبة وشارة "قريباً" للألعاب غير الجاهزة بعد.
